### Cách chạy code ?

- Bước 1: clone về
- Bước 2: cd frontend
- Bước 3: npm i
- Bước 4: npm run dev

### Lưu ý về format code !

- Cài prettier extension nếu dùng vscode giúp tui
- Thư mục .vscode tui custom để khi nhấn ctrl + s sẽ tự động format theo file .prettierrc
- ko thích cài extension thì trước khi pull request thì --> npm run format
- Tại sao lai có cái prettier này ? -> mỗi ông format mỗi kiểu đẩy lên git nó diff quá trời ! -> khó review code !
